Packages

Packages are built in the openSUSE Build Service for a variety of Linux distributions. You can find all of the packages in home:/mbarringer or use the "1 Click Install" links to automate the process a bit:

RPMs
  • openSUSE 11.4
  • openSUSE 11.3
  • openSUSE 11.2
  • openSUSE Factory
  • SLED 11 SP1
  • Fedora 14
  • Fedora 13
  • Fedora 12
  • Mandriva 2010
  • Mandriva 2010.1

DEBs
  • (K)Ubuntu 10.10
  • (K)Ubuntu 10.04
  • Debian 6.0
Not all packages are tested on all distributions for each version, so if a package doesn't work for you, let us know.

Source Code

You can browse the source code here, or if you want to download the code, you have two options:
  • Download a tarball
  • Get the latest code with git clone git://entomologist.git.sourceforge.net/gitroot/entomologist/entomologist
To compile, run qmake followed by make. You'll need Qt >= 4.4 and qjson in order to compile.
