Tracker Notes

Entomologist supports the following trackers:

  • Bugzilla 3.2+

    Entomologist will cache all updated bugs and comments on sync.

  • MantisBT

    Entomologist will cache all updated bugs on sync, but only cache comments on individual bugs when they're double-clicked in the list.

  • Launchpad

    Entomologist will cache all updated bugs on sync, but only cache comments on individual bugs when they're double-clicked in the list.

  • Trac 0.12

    Entomologist will cache all updated bugs on sync, but only cache comments on individual bugs when they're double-clicked in the list. The Trac install needs to have the XMLRPC plugin enabled, and Entomologist has not been tested with an install that uses Account Manager. Please let us know if it doesn't work for you.
