About

The goal of Entomologist is to provide a cross-platform, open source bug tracker client that works with multiple bug trackers (Bugzilla, Launchpad, etc) and provides an easier way to work with bugs on the desktop.
Currently, it will cache bugs pulled from Bugzilla (3.2+) instances and certain fields can be modified or comments added. Modified fields in the bug list will show up in red, and they can be submitted back to the bug tracker by pressing the Upload button. There are other things on the todo list. File a bug or send me an email if you have any feature requests.

Source Code

Source code is hosted on SourceForge. To get the source, you can:

git clone git://entomologist.git.sourceforge.net/gitroot/entomologist/entomologist (read-only)

Or browse it here.

To install, run qmake and make
