Source Code

Source code is hosted on SourceForge. To get the source, you can:

git clone git://entomologist.git.sourceforge.net/gitroot/entomologist/entomologist (read-only)

Or browse it here.

In order to compile, you'll need to first install libqxt version 0.6


Then run qmake and make
